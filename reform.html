Controls

Scale

Expand or shrink the selected notes. Precise values can be typed in the valuebox above the slider. The Anchor control sets the coordinate where the notes expand from/shrink to.

Curve

Redistribute selected notes' timings along a curve. The Curve Type control allows selection of 2 different curve types. The Anchor control sets this transformation to happen in either Note-Range or Selection-Range.

Shift

Shift all of the selected notes forward or backward equally.

Overflow

Use available empty columns when necessary to avoid notes being overwritten.

Condense

Use as few columns as possible without notes being overwritten.

Redistribute

Redistribute note timings evenly. The Anchor control sets this transformation to happen in either Note-Range or Selection-Range.

Collision Control

Determines which notes are overwritten in the event of note collisions. The options are to keep earlier notes, later notes, notes which are selected, and notes which are not selected.

Anchor

Determines the coordinate from which notes are Scaled, and the range within which notes are Curved and Redistributed. Note-Range: from the first to the last note in the selection. Selection-Range: from the top to the bottom of the selection box.

Volume Transform

Remap Volume-Column values to a new range, curve their distribution, and redistribute them evenly from the first note to the last.

Panning Transform

Remap Panning-Column values to a new range, curve their distribution, and redistribute them evenly from the first note to the last.

FX Amount Transform

Remap FX-Amount-Column values to a new range, curve their distribution, and redistribute them evenly from the first note to the last.
